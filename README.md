# Eye

`/lib/eye.py` A simple command line image viewer, written in SDL.

Usage: `./eye.py image_directory`


# Commands

| Key                | Command                |
| ------------------ | ---------------------- |
| up / scroll up     | Zoom In                |                                             
| down / scroll down | Zoom Out               |
| left               | Previous Image         |                                             
| right              | Next Image             |                                             
| h                  | Move Left              |                                            
| j                  | Move Up                |                                             
| k                  | Move Down              |
| l                  | Move Right             |
